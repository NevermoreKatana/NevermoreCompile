Getting Started with ShadScript

ShadScript is a lightweight scripting language designed for embedding in applications to provide users with a way to customize and extend functionality through scripting.

This guide will walk you through the basics of the ShadScript language, including syntax, variables, control flow, functions, and data structures.

Introduction

ShadScript is designed to be easy to read and write, with a syntax that is familiar to users of other scripting languages. It is interpreted at runtime, allowing for rapid iteration and experimentation.

The language is designed to be safe to run in untrusted environments, with built-in sandboxing and access controls to prevent malicious code from causing harm.

Installation

ShadScript is embedded in the application and does not require a separate installation. To use ShadScript, simply create a script file with the .shad extension and start writing code.

Variables

Variables in ShadScript are dynamically typed, meaning that the type of a variable is determined at runtime based on the value assigned to it. Variables are declared using the <span class="token keyword">let</span> keyword. 

<span class="token keyword">let</span> message = "Hello, world!";
<span class="token keyword">let</span> count = 10;
<span class="token keyword">let</span> pi = 3.14159;
<span class="token keyword">let</span> isTrue = true;
